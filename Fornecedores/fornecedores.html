Fornecedores

Neste sistema da ARCOS, os fornecedores são importantes para que sejam informados nas movimentações de outras telas do sistema.

A tela de fornecedores tem as funcionalidades de cadastrar, buscar, editar e remover um fornecedor.

Tutorial

1- Na tela inicial, aparecerá o menu com as opçes de respectivas telas, selecione Fornecedores > Fornecedor


2- Nesta tela, deverá ser informando o CNPJ da empresa, o nome, logradouro, numero, bairro, cep, cidade e estado. Também terá um campo de observação, caso tenha mais alguma coisa para informar, e logo após, clicar no botão Salvar para o fornecedor ser cadastrado. Além de cadastrar, pode ser feito uma busca de fornecedores já cadastrados, clicando no botão Buscar.


3- Tendo clicado no botão Buscar, terá um campo para pesquisar. A pesquisa pode ser filtrada pelas opções Nome e Cnpj selecionando um deles ou os dois. Logo após é só clicar no botão Buscar. Se o fornecedor estiver cadastrado aparecerá os dados logo abaixo, podendo atualizar seu cadastrado, alterando alguns campos e clicando no botão Editar. Ou, para excluir o cadastro, é só clicar no botão Remover e confirmar a exclusão. Para voltar para pagina de cadastro de fornecedor, é só clicar no botão Cancelar, que ele voltará.
