Assistência

São aquelas pessoas que pecisão de auxilio da ARCOS, ou seja pessoas que precisão de ajuda.

Tutorial

1° Entrar no sistema ARCOS com seu login e senha;

2° Ir com a seta do mouse ate a aba assistencia;

3° Clicar 2 vezes com o mouse sobre o campo assistido;

4° Preencher os campos com os seus respectivos dados;

5° Se todos os dados forem preenchidos, clique 2 vezes no botão salvar;

6° Para ter certeza que os dados foram salvos, utilize o botão buscar;

7° Ir com a seta do mouse novamente para a aba assitencia;

8° Clicar 2 vezes com o mouse sobre o campo assitencia;

9° Preencher os campos com os seus respectivos dados;

10° Se todos os dados forem preenchidos, clique 2 vezes no botão salvar;

11° Para ter certeza que os dados foram salvos, utilize o botão buscar;
